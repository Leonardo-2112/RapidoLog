# 🚚 Projeto Transportadora

Este repositório contém o desenvolvimento de uma página web para uma **transportadora**, criada como projeto de estudo utilizando tecnologias básicas do front-end.

---

## 📌 Sobre o projeto

O projeto consiste em um site de página única (**Single Page**) desenvolvido com foco em praticar conceitos fundamentais de desenvolvimento web.

A aplicação apresenta:

- Informações institucionais da transportadora  
- Serviços oferecidos  
- Área de contato  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (básico)

---

## 🎯 Funcionalidades

- Página única (SPA simples)  
- Layout responsivo básico  
- Seções organizadas (início, serviços, contato)  
- Interações com JavaScript

## 🚀 Como executar o projeto

1. Clone o repositório:
git clone https://github.com/Leonardo-2112/RapidoLog.git
